    # Nullable C# еҸҜз©әзұ»еһӢ

    C# жҸҗдҫӣдәҶдёҖдёӘзү№ж®Ҡзҡ„ж•°жҚ®зұ»еһӢпјҢnullable зұ»еһӢпјҲеҸҜз©әзұ»еһӢпјүпјҢеҸҜз©әзұ»еһӢеҸҜд»ҘиЎЁзӨәе…¶еҹәзЎҖеҖјзұ»еһӢжӯЈеёёиҢғеӣҙеҶ…зҡ„еҖјпјҢеҶҚеҠ дёҠдёҖдёӘ null еҖјгҖӮ

    дҫӢеҰӮпјҢNullable<Int32>пјҢиҜ»дҪң "еҸҜз©әзҡ„ Int32"пјҢеҸҜд»Ҙиў«иөӢеҖјдёә -2,147,483,648 еҲ° 2,147,483,647 д№Ӣй—ҙзҡ„д»»ж„ҸеҖјпјҢд№ҹеҸҜд»Ҙиў«иөӢеҖјдёә null еҖјгҖӮзұ»дјјзҡ„пјҢNullable< bool > еҸҳйҮҸеҸҜд»Ҙиў«иөӢеҖјдёә true жҲ– false жҲ– nullгҖӮ

    еңЁеӨ„зҗҶж•°жҚ®еә“е’Ңе…¶д»–еҢ…еҗ«еҸҜиғҪжңӘиөӢеҖјзҡ„е…ғзҙ зҡ„ж•°жҚ®зұ»еһӢж—¶пјҢе°Ҷ null иөӢеҖјз»ҷж•°еҖјзұ»еһӢжҲ–еёғе°”еһӢзҡ„еҠҹиғҪзү№еҲ«жңүз”ЁгҖӮдҫӢеҰӮпјҢж•°жҚ®еә“дёӯзҡ„еёғе°”еһӢеӯ—ж®өеҸҜд»ҘеӯҳеӮЁеҖј true жҲ– falseпјҢжҲ–иҖ…пјҢиҜҘеӯ—ж®өд№ҹеҸҜд»ҘжңӘе®ҡд№үгҖӮ
