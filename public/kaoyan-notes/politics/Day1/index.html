# 意识的能动性作用原理

意识的能动作用是人所特有的积极认识世界和改造世界的能力。

  • 第一，意识反应世界具有自觉性，意识活动具有目的性和计划性。

  • 第二，意识活动具有创造性，不仅反应事物的外部现象，而且反应事物的本质和规律。

  • 第三，意识活动具有指导实践、改造客观世界的作用。

  • 第四，意识具有控制人的行为和生理活动的作用。

# 人与自然的辩证关系原理

两前提 两基础 两方面

  • 自然界是人类社会形成的前提，是构成人类社会客观现实性的基础。

  • 实践是是物质世界分化为自然界和人类社会的的历史前提，又是使自然界和人类社会统一起来的基础。

必背自然规律的存在制约着人类的实践活动。人可以在认识规律，尊重规律的基础上，发挥自己的主观能动性，合理的改造自然，同时又改造自己。

延伸：当今世界出现的生态、环境、人口、资源等全球危机问题，并不单纯是自然系统内平衡关系的严重破坏，实际上也是人与自然关系的的严重失衡。
马克思认为，应到合理地调节人与自然之间的物质转换，在最无愧于和最合适人类本性的条件下进行这种物质变换。恩格斯也提出了自然界 “对人进行报复” 以及 “人类同自然的和解” 问题。随着人类改造自然的拓展和深化，人们越来越认识到正确处理人与自然的关系、实现人与自然和谐统一的重要性。
